A real-time collaborative drawing canvas built using **HTML Canvas + Socket.io**, allowing multiple users to draw together.  
Includes multiple brush tools, zooming, undo/redo, and a modern UI.

---

## Features

### Drawing Tools
- Pen  
- Pencil  
- Highlighter  
- Eraser  

### Controls
- Adjustable Brush Size  
- Color Picker  
- Undo  
- Redo  

###  Canvas Features
- Large Canvas Area
- Zoom In / Zoom Out
- Clean white background
- Smooth rendering on resize

###  Real-Time Collaboration
- Multiple users can draw simultaneously
- Live sync using Socket.io

###  UI
- Sidebar tool panel
- FontAwesome icons
- Responsive layout

---

##  Tech Stack

| Component | Technology |
|----------|------------|
| Frontend | HTML, CSS, JavaScript |
| Realtime | Socket.io |
| Backend  | Node.js + Express |
| Canvas Rendering | HTML5 Canvas API |

---

## Commands to run
 - npm install
 - npm install express socket.io
 - npm start
 - Server running at http://localhost:3000
